RF Converter setup

This plugin maps RF remote buttons to HomeKit outlets via your RF Converter V3.0 device.

Step 1 — load your remotes

Pull remotes automatically from your Safemate account, or enter them manually if you don’t have one.

Fetch from Safemate

Try without a password first. If you get errno:17, add your Safemate password — some accounts require a login session before the device list is returned.

Enter remotes manually

Add each remote, its device IP/MAC, and the available keys.

Map outlets

Each outlet appears in HomeKit. Pick which remote it controls and which key to press for ON/OFF.
